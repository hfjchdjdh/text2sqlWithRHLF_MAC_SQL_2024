# Text2SQL ——基于RLHF的多智能体部署方式

## 配置方法：

1. 创建虚拟环境并指定Python版本：

    ```bash
    conda create metagpt python=3.10
    ```

2. 激活虚拟环境：

    ```bash
    conda activate metagpt
    ```

3. 安装依赖项：

    ```bash
    conda install --file requirements.txt
    ```

## PLARF：

1. 进入PLARF目录：

    ```bash
    cd PLARF
    ```

2. 运行Web演示：

    ```bash
    python web_demo.py
    ```
